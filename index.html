Demo for Computer Graphics Lab

This repository contains demos for the course Computer Graphics at the Johannes Kepler University Linz.

Projects

  • 00_zbuffer
    Basic demo that demonstrates the depth buffer.
  • 00_blending
    Basic demo that demonstrates Alpha-Blending. The scene contains a floor and 3 quads with transparency. The blending function can be changed interactively.
  • 00_texturing
    Basic demo that shows a scene with models and a floor with textures. Textures and the model can be changed. Texture filtering techniques (i.e., mipmapping and anisotropic filtering) are demonstrated.
